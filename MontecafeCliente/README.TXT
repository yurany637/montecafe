Proyecto: Montecafe  

Módulos desarrollados:

1️⃣ Módulo Cliente  
- Registrar cliente  
- Consultar clientes  
- Actualizar clientes  
- Eliminar clientes  

2️⃣ Módulo Proveedores  
- Registrar proveedor  
- Consultar proveedores  
- Actualizar proveedores  
- Eliminar proveedores  

3️⃣ Módulo Pedido / Venta  
- Registrar pedidos por cliente  
- Solicitar producto, cantidad, valor y cliente  
- Generar resumen de ventas por cliente, producto, cantidad y valor  

4️⃣ Módulo Inventario  
- Registrar producto al inventario  
- Consultar stock  
- Actualizar stock  

5️⃣ Módulo Reportes  
- Generar resumen general del sistema (clientes, ventas, inventario)
